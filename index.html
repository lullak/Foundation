About Foundation

- Developed by ZURB, since 2019 is maintained by volunteers.

- Originally released in 2011, current version 6 released in 2015.

- Responsive framework for websites, apps and emails.

- Build mostly in HTML file using classes.

- Based on a grid system

- Building blocks: pre-made components like cards, headers, etc.
